# Inglise keel Eesti keel Seletus Näide
1 charset märgistik Märgistik on märgikoodide kogum, mis määrab, kuidas tekst tuleks kodeerida või tõlgendada, näiteks UTF-8. -
2 inline element sisse-line element Sisse-line element on HTML/CSS termin, mis kirjeldab elementi, mis paigutub rea sees ja ei võta enda alla terve rida, vaid ainult selle osa, mida ta vajab. <span>See on sisse-line element</span>
3 block element plokk-element Plokk-element on HTML/CSS termin, mis täidab kogu olemasoleva laiuse ja tavaliselt algab uuel realt, eraldades ennast teistest elementidest. <div>See on plokk-element</div>
4 padding täitevahemik Täitevahemik on CSS omadus, mis määrab elemendi sisemise tühiku elemendi ja selle piirjoone vahel. padding: 10px;
5 margin välisevahemik Välisevahemik on CSS omadus, mis määrab elemendi välise tühiku, luues seeläbi ruumi elemendi ümbritsevate elementide jaoks. margin: 20px;
6 shorthand lühend Lühend on tavaliselt kasutatav kiirem või kokkuvõtlikum viis midagi kirjeldada või esitada. margin: 10px 20px 10px 20px;
7 wireframe juhtjoon Juhtjoon on kavandatud struktuur, mis näitab lehe või rakenduse põhielemente, ilma detailide või kujunduseta. -
8 Media query meediumipäring Meediumipäring on CSS-i tehnik, mis võimaldab kohandada stiile vastavalt seadme omadustele, näiteks ekraani suurusele. @media (max-width: 600px) { /* Stiilid mobiilivaate jaoks */ }
9 hover hõljumine Hõljumine viitab seisundile, kus kasutaja suunab kursori mingi elemendi kohale, tavaliselt põhjustades mingit visuaalset muutust. a:hover { color: red; }
10 border-box piirikast Piirikast on CSS mudel, kus elemendi suurus sisaldab piirjooni ja täitevahemikku, mitte ainult sisu. box-sizing: border-box;
11 padding-box täitevahemiku kast Täitevahemiku kast on CSS mudel, kus elemendi suurus sisaldab ainult piirjooni ja täitevahemikku, mitte sisu. box-sizing: padding-box;
12 scroll kerimine Kerimine viitab lehe või elemendi liikumisele ekraanil, kui sisu ei mahu korraga kuvatavale alale. overflow: auto;
13 content stacking sisu kuhjamine Sisu kuhjamine viitab erinevate sisuüksuste paigutamisele üksteise peale, näiteks kihiliste elementide loomisele. position: relative; z-index: 2;
14 hypertext hüpertekst Hüpertekst on tekst, mis sisaldab lingitud elemente, mis võimaldavad hüpiknavigatsiooni teiste dokumentide või osade vahel. <a href="https://www.example.com">Kliki siia</a>
15 hyperweb hüpervõrk Hüpervõrk võib viidata laiemale interneti kontseptsioonile, kus seotud andmed on omavahel tugevalt seotud. -
16 viewport vaateaken Vaateaken viitab kasutatavale ekraani alale, kus veebileht või rakendus on nähtav. <meta name="viewport" content="width=device-width, initial-scale=1.0">
17 semantic markup semantiline märgistus Semantiline märgistus tähendab HTML-märgiste kasutamist nende tähenduse järgi, mitte ainult välimuse määramiseks. <header>Lehe päis</header>
18 checkbox märkeruut Märkeruut on graafiline kasutajaliidese element, mis võimaldab kasutajal valida mitmest valikust ühe või mitu. <input type="checkbox"> Vali mind
19 radio buttons raadionupud Raadionupud on graafiline kasutajaliidese element, kus kasutaja saab valida ainult ühe valiku mitme võimaluse seast. <input type="radio" name="group"> Valik 1
<input type="radio" name="group"> Valik 2
20 dropdown select rippmenüü valik Rippmenüü valik võimaldab kasutajal valida ühe võimaluse rippmenüüst. <select>
<option value="1">Valik 1</option>
<option value="2">Valik 2</option>
</select>
21 input sisend Sisend on kasutajaliidese element, mis võimaldab kasutajal sisestada teksti või muud teavet. <input type="text" placeholder="Sisesta siia">
22 disabled keelatud Keelatud tähendab, et teatud kasutajaliidese element või funktsioon ei ole hetkel aktiivne ega kasutatav. <button disabled>Klõpsa mind</button>
23 tooltip tööriip Tööriip on kasutajaliidese element, mis kuvab kontekstipõhist teavet, kui kasutaja liigub kursoriga elemendi kohale. <span title="Lisateave">Hõljuge siin</span>
24 toggle switch lülituslüliti Lülituslüliti võimaldab kasutajal vahetada kahe seisu vahel, näiteks sisse ja välja. <label class="switch">
<input type="checkbox">
<span class="slider"></span>
</label>
25 autocomplete automaatne täitmine Automaatne täitmine viitab funktsioonile, mis pakub kasutajale automaatseid soovitusi või täidab sisestuse automaatselt. <input type="text" name="country" list="countries"> <datalist id="countries"> <option value="Estonia"> <option value="Finland"> <option value="Latvia"> <option value="Lithuania"> </datalist>
26 slider liugur Liugur võimaldab kasutajal muuta väärtust loendites, liikudes liugurit üles või alla. <input type="range" min="1" max="100" value="50" id="mySlider">
27 root element juurelement Juurelement on HTML-dokumendi kõige ülemine element, millele kõik teised elemendid on alamhulgad. -
28 grid layout võrgustikuladestik Võrgustikuladestik on CSS-i omadus, mis võimaldab luua veebilehe paigutust ruudustikulise struktuuriga. .container { display: grid; grid-template-columns: auto auto auto; }
29 UI library kasutajaliidese raamatukogu Kasutajaliidese raamatukogu on kogumik eelnevalt loodud komponente ja stiile, mida saab kasutada kiireks ja järjepidevaks kasutajaliidese arendamiseks. -
30 sidebar küljeriba Küljeriba on lehe või rakenduse külgmine ala, kus tavaliselt asuvad täiendavad navigeerimise või kontrolli elemendid. <div class="sidebar"> <p>Navigatsioon</p> <a href="#home">Avaleht</a> <a href="#about">Teave</a> <a href="#contact">Kontakt</a> </div>
